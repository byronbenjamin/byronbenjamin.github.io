Week 4

January 3, 2015

This week in DBC we focused on Arrays and Hashes. Arrays and Hashes are popular structures in Ruby used for storing data.

Arrays can store any type of thing that we have access to in Ruby. That could be a string, a number, or even another array.

A hash is similar to an array. Hashes are basically the same as arrays, except that a hash not only contains values, but also keys pointing to those values. Each key can occur only once in a hash.

Here is an example of a array:

numbers = [1,2,"three",[4,5,6]]

Here is an example of a hash:

grocery_list{
"milk" => "chocolate",
"cookies" => "oreos",
"cereal" => "cheerios"
}

While learning about the two I found a great a example to explain the two. It was explained something like this. If you make a grocery list into an array you can only add a single item. In a hash you can name the item it is that you are looking for, which would be called the key and you can also add the brand name, the type, or the quantity, which would be the value.
